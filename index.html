How the Microsoft Graph API Integration Works
✉
Reading Outlook Mail
Uses GET /me/messages with $filter and $search to find emails with PDF attachments. Attachment content fetched via GET /me/messages/{id}/attachments.
📊
Writing to Excel
Uses POST /me/drive/items/{id}/workbook/worksheets/{sheet}/range to append rows directly into your Excel file on OneDrive — no download needed.
📁
Filing PDFs to OneDrive
Uses PUT /me/drive/root:/{path}/{filename}:/content to upload PDF bytes directly into your OneDrive folder, organised by category and date.
