OneDrive Auto-Sync Script

Save this PowerShell script on your Windows PC to automatically move downloaded reports into OneDrive:

# Auto-move Financial Reports to OneDrive
$downloads = "$env:USERPROFILE\Downloads"
$onedrive  = "$env:USERPROFILE\OneDrive\Financial Reports"

if (!(Test-Path $onedrive)) { New-Item -ItemType Directory -Path $onedrive }

$watcher = New-Object IO.FileSystemWatcher $downloads
$watcher.Filter = "Financial_Report_*.xlsx"
$watcher.EnableRaisingEvents = $true

Register-ObjectEvent $watcher Created -Action {
  $file = $Event.SourceEventArgs.FullPath
  Start-Sleep 2
  Move-Item $file $onedrive -Force
  Write-Host "Filed to OneDrive: $($Event.SourceEventArgs.Name)"
}

Save as .ps1 and run in PowerShell. Requires OneDrive desktop app installed.
